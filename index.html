Hexo Disqus 設定

發表於 2021-10-04 分類於 Hexo 教學 Disqus：

Disqus

介紹

我想一個網誌另外一格重要的功能大概就是留言板，為了能夠跟人互動可能就需要加入些套件來做使用

那這邊我介紹一個蠻多網誌都有使用的 Disqus

使用教學

1.先到網站進行註冊，註冊完畢應該會出現以下畫面，選擇下面那個一個選項

2.在Website Name輸入你想設定的名稱，然後選擇Category後送出

3.選擇左邊的第二個Install Disqus

4.選擇圖中方框全起來的地方，選擇後應該會跳到下個畫面拉到最下面選擇Configure

5.這裡可以直接Next有需要再回來填寫

6.這邊依照自己的需求來做選擇，我是選擇上面那個送出

7.大致上就設定完畢了

找到Dismiss Setup選擇這個即可

8.打開Settings -> general -> 找到 Shortname 複製起來

9.這邊以NexT的Theme作介紹，找到 theme\next 的資料夾裡的 _config.yml 找到以下的參數做修改
10.完成，啟動去看看有沒有成功吧！

What is "this" in JavaScript

發表於 2021-10-02 更新於 2021-10-04 分類於 JavaScript Disqus：

什麼是this

  • this 是 JavaScript 的一個關鍵字。
  • this 是 function 執行時，自動生成的一個內部物件。
  • 隨著 function 執行場合的不同，this 所指向的值，也會有所不同。
  • 在大多數的情況下，this 會因為 function 的呼叫方式而有所不同。
