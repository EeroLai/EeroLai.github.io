Hexo Admin 貼上圖片位置失效

發表於 2021-10-04 更新於 2023-02-22 分類於 Hexo 教學 Disqus：

介紹

Hexo Admin圖片貼上後自動產生的路徑一直出現破圖

路徑也真的看不出到底有甚麼錯誤，看了看最後找到了這個解決方法

設定與安裝

1.找到hexo/_config.yml裡面的post_asset_folder: false改成true

2.安裝 npm install hexo-asset-image --save

路徑修改

截圖後再Hexo Admin編輯區貼上

經過以上調整圖片應該就能正常顯示了
